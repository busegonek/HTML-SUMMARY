  -----------------------------------ENGLISH VERSION-----------------------------------

1-> What is W3C?

W3C, or the World Wide Web Consortium, is an organization that sets the "standards" for the internet. It was founded in 1994 by Tim Berners-Lee. Its purpose is to create various rules and standards to ensure the development and longevity of the web.

For example, it defines how web technologies like HTML and CSS should function. These standards ensure that websites look and work the same across different browsers and devices.

In short, W3C is a group of experts and organizations working to make the web better and more accessible for everyone.

HTML contains tags, and these tags have a standard that has been previously specified by W3C.

2-> What is W3C Validator / What is it Used For?

W3C Validator is a tool that checks the HTML, XHTML, or CSS code of web pages. W3C (World Wide Web Consortium) is an organization that sets web standards, and this validator allows web developers to determine whether their code complies with those standards.

Here’s the link:

https://validator.w3.org/

Validate by URL: You can check by providing the online address of the site.
Validate by File Upload: You can check by uploading a file.
Validate by Direct Input: You can check by pasting the code directly.
